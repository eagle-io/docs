Attachment
==========
|icon-attachment| Attachments include images, documents and other files stored in `eagle.io <https://eagle.io>`_. New attachments can be manually uploaded by a user or acquired via an :ref:`Attachment Source <node-configuration-attachmentsource>`. Attachments can be added inside Workspaces, Locations, Groups and Attachment Sources.

To view attachments navigate to :menuselection:`List View --> Attachments`. You will see a list of all attachments within the Selected Node. Requires *view attachments* permission.

To upload attachments click the *upload attachments* button from the :ref:`attachments list view <view-list-attachments>` toolbar or 
drag and drop files directly from your desktop into the browser. Attachments will be created in the currently Selected Node.
