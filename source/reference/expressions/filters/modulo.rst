.. _liquid-filters-modulo:

modulo
=======

Returns the remainder of a division operation.

Input

.. code:: liquid

   {{ 3 | modulo: 2 }}

Output

.. code:: text

   1

Input

.. code:: liquid

   {{ 24 | modulo: 7 }}

Output

.. code:: text

   3

Input

.. code:: liquid

   {{ 183.357 | modulo: 12 }}

Output

.. code:: text

   3.3569999999999993
