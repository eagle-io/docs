.. _liquid-filters-times:

times
=======

Multiplies a number by another number.

Input

.. code:: liquid

   {{ 3 | times: 2 }}

Output

.. code:: text

   6

Input

.. code:: liquid

   {{ 24 | times: 7 }}

Output

.. code:: text

   168

Input

.. code:: liquid

   {{ 183.357 | times: 12 }}

Output

.. code:: text

   2200.284
