.. _liquid-filters-plus:

plus
=====

Adds a number to another number.

Input

.. code:: liquid

   {{ 4 | plus: 2 }}

Output

.. code:: text

   6

Input

.. code:: liquid

   {{ 16 | plus: 4 }}

Output

.. code:: text

   20

Input

.. code:: liquid

   {{ 183.357 | plus: 12 }}

Output

.. code:: text

   195.357
