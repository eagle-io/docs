.. _liquid-filters-minus:

minus
======

Subtracts a number from another number.

Input

.. code:: liquid

   {{ 4 | minus: 2 }}

Output

.. code:: text

   2

Input

.. code:: liquid

   {{ 16 | minus: 4 }}

Output

.. code:: text

   12

Input

.. code:: liquid

   {{ 183.357 | minus: 12 }}

Output

.. code:: text

   171.357
