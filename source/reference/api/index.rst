HTTP API
========

.. image:: /_static/coming-soon.png

.. note:: 
   We are currently developing a full-featured HTTP API which will make it easy for developers to interact with eagle.io from a 3rd party application or service. We intend to do the initial release early 2014.

Feature Highlights
------------------

- Node management including: Create, Read, Update, Delete
- Operate on Sources: Acquire, Control, Import
- Historic queries and data retrieval
- Event queries and Alarm Management
- User Management and Sharing
