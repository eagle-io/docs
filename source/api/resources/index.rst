.. _api-resources:

API Resources
==============

API Resources are the URL endpoints for interacting with the system.
Interactions may include creating, reading, updating or deleting an asset, extracting or inserting historic data, or performing operations such as triggering data acquisition, control operations or device configuration. 

Refer to the :ref:`Quick Start <api-overview>` guide for instructions on using the API.
